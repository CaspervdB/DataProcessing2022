Voor deze API moet node js geinstalleerd zijn op de computer.

De API kan via onderstaande methode worden gestart.
Navigeer naar de "Deel 2" map en open daar een terminal.
Voer in de terminal het "nodemon index.js" commando uit.

Na het starten van de API zal hij te bereiken zijn op "http://localhost:3000/" tenzij er een andere port in de systeem variablen is ingesteld.
